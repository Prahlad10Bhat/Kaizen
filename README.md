## About

Kaizen is an all-in-one productivity application built to replace the need for multiple disconnected apps.

Instead of switching between note-taking apps, to-do lists, calendars, habit trackers, timers, workout apps, and utility tools, Kaizen brings everything together in one fast, modern workspace.

Designed with an offline-first philosophy, Kaizen gives you full control over your data while providing a clean and distraction-free experience.

Kaizen is designed with privacy in mind.

✔ Local data storage

✔ No ads

✔ No tracking

✔ You own your data

Future versions will include optional account sync for users who want access across multiple devices.
